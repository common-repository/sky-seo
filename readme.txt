Improve your WordPress SEO: Write better content and have a fully optimized WordPress site using Sky SEO plugin.

== Description ==

This plugin helps you optimize the display of  <title>, description, keywords on search tools like google, bing, yahoo and enhance rank of your keywords. It is very easy , not complicated and does not have too many functions as those of Yoast seo

== Installation ==

1. Upload `sky-seo.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Sky Seo' menu in WordPress
